# Email Summary Frontend

Frontend aplikacji do podsumowywania emaili, zbudowany w React + Vite + TailwindCSS.

## Instalacja

```bash
npm install
```

## Uruchomienie

```bash
npm run dev
```

Aplikacja będzie dostępna pod adresem: http://localhost:3000

## Build produkcyjny

```bash
npm run build
```

## Wymagania

- Node.js >= 16
- Backend API uruchomiony na http://localhost:8000

## Funkcjonalności

- Wklejanie JSON z emailami
- Generowanie podsumowania przy użyciu Claude API
- Loading state podczas generowania
- Obsługa błędów
- Minimalistyczny, nowoczesny design
