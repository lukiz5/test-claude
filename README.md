# Hello Claude Project

A simple Python project with a hello() function.

## Installation

No additional dependencies required. You only need Python 3 installed on your system.

## Usage

To run the program:

```bash
python main.py
```

This will execute the `hello()` function and print "Hello, Claude" to the console.

## Using the function in your code

You can also import and use the `hello()` function in your own Python code:

```python
from main import hello

message = hello()
print(message)  # Output: Hello, Claude
```
